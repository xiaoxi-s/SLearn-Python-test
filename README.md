# SLearn-Python-test
Test SLearn on Python end

To run the test on your own computer, you need these pacakges/software:

[cmake](https://cmake.org/cmake/help/v3.17/guide/tutorial/index.html?highlight=install#installing-and-testing-step-4)
[gcc](https://gcc.gnu.org/)
[pybind11](https://github.com/pybind/pybind11)
[Eigen3](http://eigen.tuxfamily.org/index.php?title=Main_Page)
[pytest](https://github.com/pytest-dev/pytest)
[SLearn](https://github.com/TheCommonOne/SLearn.git)

---

## Procedures

Assume cmake and gcc are properly installed and added to system path.

1. Install pybind11 and Eigen3
2. Install SLearn
3. Install pytest
4. Run pytest in the root directory of this repository.
